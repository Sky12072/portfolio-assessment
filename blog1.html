Coding Journey

Oct 18, 2022   –   3min read

I started my coding journey as a student working towards becoming a web developer. I was always hard-working and motivated to learn, and my coding journey has been an amazing experience so far.

I've learnt so much and continue to grow as a developer every day. I'm extremely grateful for everything that I've learnt in my coding journey in the last 1.5 years, and I can't wait to see what the future holds. I'm thankful to Coder Academy which helped me every step of the process to be where I am now.

Some of the biggest challenges I've faced so far have been learning how to effectively use different programming languages and frameworks and understanding how to apply what I've learnt to real-world situations. However, these challenges have also been some of the most rewarding experiences, as I've been able to overcome them and grow as a developer. I'm excited to see what challenges lie ahead in my coding journey.

I have written other blogs which also played a bigger role internally to help me to be a better coder, and if you're a curious developer like I am, feel free to find out the rest of the four blogs!
